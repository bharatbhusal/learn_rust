HashMap

HashMaps:

  1. Definition:
    • A HashMap is a collection type in Rust that stores key-value pairs.
    • It provides fast lookup and insertion of values based on their keys.
    • HashMaps are implemented as hash tables, which make them efficient for large datasets.
  1. Declaration and Initialization:
    • You can declare a HashMap using the HashMap type from the standard library.
    • HashMaps require importing the HashMap type from the std::collections module.
    • You can initialize a HashMap using the HashMap::new() method.
    • Here's an example of declaring and initializing a HashMap:
      use std::collections::HashMap;

let mut scores: HashMap<String, i32> = HashMap::new(); // Empty HashMap
  1. Adding and Updating Entries:
    • You can add entries to a HashMap using the insert method, which takes a key-value pair.
    • The key and value must be of the specified types when declaring the HashMap.
    • If the key already exists, the corresponding value will be updated.
    • Here's an example of adding and updating entries in a HashMap:
      use std::collections::HashMap;

let mut scores: HashMap<String, i32> = HashMap::new();

scores.insert(String::from("Alice"), 100);
scores.insert(String::from("Bob"), 90);

scores.insert(String::from("Bob"), 95); // Updating the value for the "Bob" key
  1. Accessing and Removing Entries:
    • You can access entries in a HashMap using the get method, which takes the key as an argument and returns an Option containing the value.
    • The remove method allows you to remove entries from the HashMap, given the key.
    • Here's an example of accessing and removing entries from a HashMap:
      use std::collections::HashMap;

let mut scores: HashMap<String, i32> = HashMap::new();

scores.insert(String::from("Alice"), 100);
scores.insert(String::from("Bob"), 90);

if let Some(score) = scores.get("Alice") {
    println!("Alice's score: {}", score);
}

scores.remove("Bob"); // Removing the entry for the "Bob" key
  1. Iterating over Entries:
    • You can iterate over the entries of a HashMap using a for loop or iterator methods.
    • The for loop with destructuring allows you to access both the key and value of each entry.
    • The iter method returns an iterator over the entries of the HashMap.
    • Here's an example of iterating over the entries of a HashMap:
      use std::collections::HashMap;

let mut scores: HashMap<String, i32> = HashMap::new();

scores.insert(String::from("Alice"), 100);
scores.insert(String::from("Bob"), 90);

for (name, score) in &scores {
    println!("{}: {}", name, score);
}
  1. Checking Existence of a Key:
    • You can check whether a key exists in a HashMap using the contains_key method.
    • The method returns a boolean indicating whether the key is present or not.
    • Here's an example of checking the existence of a key in a HashMap:
      use std::collections::HashMap;

let mut scores: HashMap<String, i32> = HashMap::new();

scores.insert(String::from("Alice"), 100);

 if scores.contains_key("Alice") {
     println!("Alice's score exists!");
 }
