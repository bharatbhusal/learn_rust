Arrays

  1. What is an array in Rust? An array in Rust is a fixed-size, contiguous collection of elements of the same type. The size of the array is determined at compile time and cannot be changed during runtime.
  1. Creating an array: You can create an array using square brackets [] and explicitly specify the type and size of the array. For example:
    let numbers: [i32; 5] = [1, 2, 3, 4, 5];
  1. Array elements and indexing: Array elements are accessed using zero-based indexing. You can access an element by specifying its index within square brackets []. For example:
    let numbers: [i32; 5] = [1, 2, 3, 4, 5];
let first_number = numbers[0]; // Accessing the first element
  1. Array length: The length of an array is known at compile time and can be obtained using the .len() method. For example:
    let numbers: [i32; 5] = [1, 2, 3, 4, 5];
let length = numbers.len();
  1. Array initialization: You can initialize an array with the same value for all its elements using the Array::fill() method. For example:
    let numbers = [0; 5]; // Initializes the array with five 0s
  1. Iterating over an array: You can iterate over the elements of an array using a for loop or by calling the .iter() method to obtain an iterator. For example:
    let numbers: [i32; 5] = [1, 2, 3, 4, 5];

// Using a for loop
for num in &numbers {
    println!("{}", num);
}

// Using an iterator
for num in numbers.iter() {
    println!("{}", num);
}
  1. Array comparison: Arrays can be compared for equality using the == and != operators. The elements of the arrays are compared pairwise. For example:
    let array1 = [1, 2, 3];
let array2 = [1, 2, 3];

if array1 == array2 {
    println!("Arrays are equal");
}
  1. Array operations: Arrays in Rust have limited built-in operations. However, you can perform common operations such as sorting, searching, and filtering by using methods from the Rust standard library or external crates.
  1. Array and ownership: Unlike vectors (Vec), arrays in Rust have a fixed size and are stored on the stack rather than the heap. They are owned and reside entirely within a single stack frame.
  1. Array and borrowing rules: Arrays follow Rust's borrowing rules, allowing you to borrow immutably or mutably. You can borrow the entire array or borrow specific elements using slices (&[T]).

Arrays provide a simple and efficient way to store a fixed number of elements of the same type. They are commonly used when you know the size of the collection at compile time and want to ensure a fixed layout in memory.
