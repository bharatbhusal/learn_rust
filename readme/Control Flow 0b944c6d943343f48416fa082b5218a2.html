Control Flow

Conditional Statements:

  1. If Statements:
    • The if statement allows you to conditionally execute a block of code based on a boolean expression.
    • The basic syntax of an if statement in Rust is as follows:
      if condition {
    // Code to execute if the condition is true
} else {
    // Code to execute if the condition is false
}
    • The else clause is optional. It allows you to provide an alternative code path when the condition is false.
  1. Else If Statements:
    • You can use the else if clause to chain multiple conditions together. It allows you to test additional conditions if the previous ones are false.
    • Here's an example of using else if in an if statement:
      if condition1 {
    // Code to execute if condition1 is true
} else if condition2 {
    // Code to execute if condition2 is true
} else {
    // Code to execute if both condition1 and condition2 are false
}
  1. Nested If Statements:
    • You can nest if statements inside other if statements to create more complex conditional logic.
    • Here's an example of nesting if statements:
      if condition1 {
    if condition2 {
        // Code to execute if both condition1 and condition2 are true
    } else {
        // Code to execute if condition1 is true and condition2 is false
    }
} else {
    // Code to execute if condition1 is false
}

Iterative Statements:

  1. Loop Statement:
    • The loop statement allows you to create an infinite loop that continues until explicitly stopped using the break keyword.
    • Here's an example of using a loop statement:
      loop {
    // Code to execute repeatedly

    if condition {
        break; // Exit the loop
    }
}
  1. While Statement:
    • The while statement repeats a block of code as long as a condition is true.
    • The syntax of a while statement in Rust is as follows:
      while condition {
    // Code to execute as long as the condition is true
}
    • The loop will continue as long as the condition remains true. Once the condition evaluates to false, the loop will terminate, and program execution will continue with the next statement.
  1. For Statement:
    • The for statement is used to iterate over collections, ranges, or any type that implements the Iterator trait.
    • Here's an example of using a for statement:
      let numbers = [1, 2, 3, 4, 5];

for number in numbers.iter() {
    // Code to execute for each element in the array
}
    • The for statement assigns each element of the collection to the loop variable (number in the example) and executes the code block for each iteration.
  1. Range-Based Iteration:
    • Rust provides a range syntax (start..end) that represents a sequence of values from start to end-1.
    • Here's an example of using a range in a for statement:
      for number in 1..=5 {
    // Code to execute for each value in the range (1, 2, 3, 4, 5)
}

  • The ..= operator is used to include the end value of the range.
  1. Loop Control Statements:
    • Inside iterative statements, you can use control flow statements like break and continue to modify the flow of execution.
    • The break statement is used to exit the loop prematurely and continue with the next statement after the loop.
    • The continue statement is used to skip the rest of the current iteration and move to the next iteration.

Control flow statements, including conditional statements and iterative statements, are fundamental in programming and allow you to make decisions and repeat actions based on conditions. Understanding these concepts will help you write more dynamic and flexible code in Rust.
