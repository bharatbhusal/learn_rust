Vector

Vectors:

  1. Definition:
    • A vector is a dynamic, growable array in Rust.
    • Vectors are a collection type that allows you to store multiple values of the same type.
    • Vectors are allocated on the heap and can grow or shrink as needed.
  1. Declaration and Initialization:
    • You can declare a vector using the Vec keyword followed by angle brackets < > and the type of the elements.
    • Vectors can be initialized using the vec! macro, which takes a comma-separated list of initial values.
    • Here's an example of declaring and initializing a vector:
      let mut numbers: Vec<i32> = Vec::new(); // Empty vector

let names = vec!["Alice", "Bob", "Charlie"]; // Vector with initial values
  1. Adding Elements:
    • You can add elements to a vector using the push method, which appends the element at the end of the vector.
    • The vector must be mutable (mut) to modify its contents.
    • Here's an example of adding elements to a vector:
      let mut numbers: Vec<i32> = Vec::new();

numbers.push(10);
numbers.push(20);
numbers.push(30);
  1. Accessing Elements:
    • You can access individual elements of a vector using square brackets ([]) and the index of the element.
    • Rust uses zero-based indexing, so the first element is at index 0, the second element is at index 1, and so on.
    • Here's an example of accessing elements in a vector:
      let numbers = vec![10, 20, 30];

let first_number = numbers[0];
let second_number = numbers[1];
  1. Updating Elements:
    • You can update elements in a vector by assigning a new value to a specific index.
    • The vector must be mutable (mut) to modify its contents.
    • Here's an example of updating elements in a vector:
      let mut numbers = vec![10, 20, 30];

numbers[1] = 25;
  1. Removing Elements:
    • You can remove elements from a vector using the remove method, which takes the index of the element to remove.
    • The method returns the removed element.
    • Here's an example of removing an element from a vector:
      let mut numbers = vec![10, 20, 30];

let removed_number = numbers.remove(1); // Remove the element at index 1 (20)
  1. Iterating over Elements:
    • You can iterate over the elements of a vector using a for loop or iterator methods.
    • The for loop iterates over each element of the vector sequentially.
    • Here's an example of iterating over the elements of a vector:
      let numbers = vec![10, 20, 30];

for number in &numbers {
    println!("Number: {}", number);
}
  1. Vector Length and Capacity:
    • The length of a vector is the number of elements it currently contains, which can be obtained using the len method.
    • The capacity of a vector is the total number of elements it can hold without reallocation, which can be obtained using the capacity method.
    • Here's an example of

accessing the length and capacity of a vector: ```rust let numbers = vec![10, 20, 30];

 let length = numbers.len();
 let capacity = numbers.capacity();
 ```
