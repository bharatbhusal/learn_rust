Characters

Here's an overview of characters in Rust for newcomers:

  1. What is a character in Rust? In Rust, a character represents a Unicode scalar value, which can be any Unicode character ranging from basic Latin letters and digits to emojis and other international characters. Rust uses the char type to represent characters.
  1. Character literals: Character literals are written using single quotes '. For example, 'A', '😀', or '9' are valid character literals in Rust.
  1. Creating a character: You can assign a character to a variable using the char type annotation and the character literal. For example:
    let letter_a: char = 'A';
let smiley_face: char = '😀';
  1. Character properties: Characters have various properties and methods associated with them. You can check if a character is alphabetic, numeric, uppercase, lowercase, or whitespace using methods like is_alphabetic(), is_numeric(), is_uppercase(), is_lowercase(), and is_whitespace(). For example:
    let ch: char = 'A';
if ch.is_alphabetic() {
    println!("It's an alphabetic character!");
}
  1. Character conversion: You can convert a char to its Unicode scalar value using the .to_digit() method, and vice versa using the std::char::from_digit() function. For example:
    let ch: char = '9';
let digit_value = ch.to_digit(10);  // Converts the character '9' to the digit value 9
  1. Character operations: Characters support various operations, such as comparison, concatenation, and iteration. You can compare characters using the standard comparison operators (==, !=, <, >, etc.). Characters can be concatenated using the + operator. You can iterate over a string to access individual characters using the .chars() method. For example:
    let ch1: char = 'A';
let ch2: char = 'B';
if ch1 < ch2 {
    println!("{} comes before {}", ch1, ch2);
}
  1. Character and string interoperability: Characters can be converted to strings using the .to_string() method, and strings can be converted to characters using the .chars() iterator. For example:
    let ch: char = 'A';
let ch_str = ch.to_string();  // Converts the character 'A' to a string "A"

let s = "Hello";
let first_char = s.chars().next();  // Retrieves the first character 'H'
  1. Character and Unicode support: Rust fully supports Unicode characters, including characters from different scripts, emojis, and special symbols. Rust's char type represents Unicode scalar values, which allows you to work with characters from any Unicode block.

Understanding how to work with characters in Rust is important for handling text and performing string manipulation. Rust's strong support for Unicode ensures that you can work with a wide range of characters and build applications that handle different languages and scripts effectively.
