Integers

  1. What are integers in Rust? Integers are whole numbers without a fractional part. Rust provides several integer types with different ranges and sizes. The most commonly used integer types are i8, i16, i32, i64, u8, u16, u32, and u64.
  1. Signed and unsigned integers: Integer types starting with i (e.g., i8, i16, i32, i64) are signed integers, which can represent both positive and negative numbers. Unsigned integers, starting with u (e.g., u8, u16, u32, u64), can only represent non-negative numbers.
  1. Creating integers: Integers can be assigned to variables using the appropriate type annotation (i8, i16, i32, i64, u8, u16, u32, u64) and a numeric literal. For example:
    let x: i32 = 42;
let y = 123; // Type inference infers the type as i32
  1. Integer literals: Integer literals can be represented in decimal form, hexadecimal form (with the prefix 0x), octal form (with the prefix 0o), or binary form (with the prefix 0b). For example, 42, 0xAB, 0o77, and 0b1010 are valid integer literals in Rust.
  1. Integer range and size: The size of an integer type determines the range of values it can represent. For example, i8 can represent integers from -128 to 127, while i16 can represent integers from -32,768 to 32,767. The larger the size of the integer type, the wider the range of representable values.
  1. Integer operations: Integers support standard arithmetic operations such as addition (+), subtraction (), multiplication (), and division (/). For example:
    let x: i32 = 42;
let y: i32 = 10;
let sum = x + y;
let product = x * y;
  1. Integer overflow and underflow: Rust provides built-in protection against integer overflow and underflow. In debug builds, if an operation causes overflow or underflow, Rust will panic and terminate the program. In release builds, Rust wraps the result using two's complement wrapping behavior.
  1. Integer conversion: Rust provides methods to convert between different integer types. You can use methods like .into() or the as keyword to explicitly convert between integer types. For example:
    let x: u8 = 42;
let y: i32 = x.into(); // Convert u8 to i32
let z: u16 = x as u16; // Convert u8 to u16
  1. Bitwise operations: Integers support bitwise operations such as bitwise AND (&), bitwise OR (|), bitwise XOR (^), bitwise complement (!), left shift (<<), and right shift (>>). These operations manipulate the individual bits of integers.
  1. Integer and control flow: Integers play a crucial role in control flow statements. They are used to control the execution of loops (while, for), conditional branches (if, `

else if, else`), and other control flow constructs in Rust.

Integers are widely used for counting, indexing, and performing arithmetic operations in programming. Understanding the properties and limitations of integer types is essential for writing correct and efficient code in Rust.
