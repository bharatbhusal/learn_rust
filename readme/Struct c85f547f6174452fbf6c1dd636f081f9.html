Struct

Structs:

  1. Definition:
    • A struct (short for structure) is a custom data type that allows you to define your own composite types.
    • It allows you to group multiple values together into a single entity with its own fields.
  1. Declaration:
    • You can declare a struct using the struct keyword followed by the struct's name and a block defining its fields.
    • Each field has a name and a type.
    • Here's an example of a struct declaration:
      struct Person {
    name: String,
    age: u32,
    is_employed: bool,
}
  1. Instantiation:
    • To create an instance of a struct, you can use the struct's name followed by curly braces and provide values for each field.
    • Here's an example of instantiating a Person struct:
      let person = Person {
    name: String::from("Alice"),
    age: 25,
    is_employed: true,
};
  1. Accessing Fields:
    • You can access individual fields of a struct using dot notation (.) followed by the field name.
    • Here's an example of accessing fields of the person instance:
      println!("Name: {}", person.name);
println!("Age: {}", person.age);
println!("Employment Status: {}", person.is_employed);
  1. Updating Fields:
    • Struct fields can be mutable, allowing you to modify their values after instantiation.
    • You can use the dot notation and assignment (=) to update individual fields.
    • Here's an example of updating the age field of the person instance:
      person.age = 30;
  1. Methods:
    • Structs can have associated functions and methods, similar to functions, but defined within the context of the struct.
    • Methods are defined using the impl keyword followed by the struct's name.
    • Here's an example of adding a method to the Person struct:
      impl Person {
    fn say_hello(&self) {
        println!("Hello, my name is {}", self.name);
    }
}
    • The &self parameter in the method signature refers to an immutable reference to the struct instance.
  1. Tuple Structs:
    • Rust also supports tuple structs, which are structs without named fields.
    • Tuple structs are declared similarly to tuples but with a name.
    • Here's an example of a tuple struct declaration and instantiation:
      struct Point(f32, f32);

let point = Point(2.5, 3.5);
    • Tuple structs can be useful when you want to group values together but don't require named fields.

Structs provide a way to define custom data types with their own fields and behavior. They are widely used in Rust for organizing and modeling data in a structured manner. By using structs, you can create more complex and flexible data structures to suit your program's needs.
